
VoiceAutoTool(仮) 
------------------------------------------------------------
Davinci Resolveで
ローカルの指定されたフォルダから音声ファイルをを引っ張ってくるツール：AutoTool_MoveSound.lua
エディットページに置かれた音声ファイルを基に字幕ファイルを自動生成するツール：AutoTool_Createset.lua
が入ってます

-------------------------------------------------------------
☆インストール方法
・VoiceAutoToolフォルダをしたのフォルダにそのまま置く
　C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\Comp

・Luaフォルダをしたのフォルダにそのまま置く
　C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Modules

以上。
本スクリプトをインストールした状態でDavinci Resolveを開き適当なプロジェクトを開いて、上のタブから
「ワークスペース→Script→Comp→VoiceAutoTool」でインストールしたスクリプトにアクセスできるはずです

--------------------------------------------------------------
☆使うための前提条件
・指定のフォルダに「音声ファイル」と「音声をテキスト化したテキストファイル」のどちらも入っている必要があります。
　VOICEROIDを使ってる人は、VOICEROIDを起動して「ツール→オプション」のとこに「テキストファイルを音声ファイルと一緒に保存する」
　にチェックが入ってれば大丈夫です。
　それ以外はしらないです。

・Davinci Resolveの設定から「スクリプトの実行の許可」で設定が必要な可能性があります。
　たぶんデフォルトだと「ローカル環境のみ実行」になってると思うから大丈夫だとは思うけど・・・

--------------------------------------------------------------
☆手順：AutoTool_MoveSound
1.とりあえず設定を変更します
「ワークスペース→Script→Comp→VoiceAutoTool→AutoTool_Config」を選択

開かれたウィンドウの「VOICEFOLDER_PATH」に、移動したい音声ファイルがおかれているフォルダを指定します
他のとこはまあなんでもいいです

2.AutoTool_MoveSoundを実行します
「ワークスペース→Script→Comp→VoiceAutoTool→AutoTool_MoveSound」を選択

なんか勝手に音声が呼び出されるはずで、Davinci Resolveのメディアプールにも音声ファイルがおかれるはずです

---------------------------------------------------------------
☆手順：AutoTool_CreateSrt
1.必要であれば設定を変更します。
エディットページのAudioのA1のとこ(一番上)を字幕化したい場合は設定を開く必要なし

2.AutoTool_Createsrtを実行します
「ワークスペース→Script→Comp→VoiceAutoTool→AutoTool_Createsrt」を選択

上手くいけばメディアプールに「srt_9384028402.srt」みたいなファイルが出現するはずです

3. 2で出力されたsrtファイルを、Davinci Resolveのエディットページのタイムラインにドラッグ＆ドロップ

----------------------------------------------------------------
☆トラブルシューティング的なQ＆A
Q. いちいちUI上でスクリプト指定するのめんどうくさい
A. Resolveにはショートカット登録機能があるので、それを使えば1操作で簡単にスクリプトを実行できます
　 （完全自動化はできたらする。できたら）

Q.字幕のテキストが「nil」になる
A.指定のフォルダに「音声をテキスト化したテキストファイル」が無かった、もしくは名前が違う

Q.Configが読み込めないって言われてる気がする
A.最悪↓にあるConfigを消してOpenConfig実行してください、リフレッシュされるはずです
　C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Config

Q.なんかうごかない
A.「ワークスペース→コンソール」を開いた状態でスクリプトを動かすとエラーメッセージが出力されるはずです
　そのエラーメッセージを作者宛に連絡くれるとうれしいです

Q.エラーだのバグだの要望だの連絡したい
A.DiscordかTwitterでおねがいします。Twitterが一番手っ取り早いです　→　@nu_ro_ku
